# El Compa Chicharrones

Este repositorio contiene el código fuente para la página web de "El Compa Chicharrones", un negocio dedicado a la venta de chicharrones y otros productos de cerdo con auténtico sabor dominicano. La página permite a los clientes explorar el menú y realizar pedidos a través de WhatsApp.

## Características

*   **Menú Interactivo:** Muestra los productos disponibles con descripciones y precios.
*   **Pedidos por WhatsApp:** Facilita la comunicación directa con el negocio para realizar pedidos.
*   **Diseño Responsivo:** Adaptado para una visualización óptima en diferentes dispositivos (móviles y escritorio).
*   **Galería de Productos:** Incluye imágenes y videos de los productos.

## Instalación

Para configurar y ejecutar la página web de "El Compa Chicharrones" en tu entorno local, sigue estos pasos:

1.  **Clonar el repositorio:**

    ```bash
    git clone https://github.com/Stiwall/elcompa_chicharron.git
    cd elcompa_chicharron
    ```

2.  **Abrir en el navegador:**

    Dado que es una página estática, simplemente abre el archivo `index.html` en tu navegador web preferido.

    ```bash
    open index.html
    ```

## Uso

Una vez abierta la página en tu navegador, podrás:

*   Navegar por las diferentes secciones del menú.
*   Ver los detalles de cada producto.
*   Hacer clic en el botón de WhatsApp para iniciar un pedido o consulta.
